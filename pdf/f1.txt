Bootstrap  представляет  собой  свободный  фреймворк  интерфейсов  для  быстрой 
и простой  Web  разработки.  Bootstrap  предоставляет  шаблоны  дизайна,  основанные 
на HTML и CSS для разметки, форм, кнопок, таблиц, навигации, диалоговых окон, каруселей 
изображений и многого другого, а также дополнительные плагины JavaScript. На основе 
Bootstrap  можно  легко  создавать  сайты  с «отзывчивым  дизайном»,  одинаково  хорошо 
выглядящие  на всех  типах  устройств,  от небольших  телефонов  до настольных 
компьютеров.
